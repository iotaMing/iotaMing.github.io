hexo博客图片无法显示

​ 原来就是因为文章中图片无法正常显示而选择了删除仓库重新搭建，但是在这次重新搭建的过程中还是遇到了这种问题，不过还好的是这次用了一个晚上还是解决了问题，但是还是觉得有必要记录一下解决办法以防万一。这种办法很玄学，正常我们使用下面第一条命令安装hexo-asset-image插件就会导致图片无法显示(后面知道是插件会把图片的路径解析错误)，当改成第二条指令安装时就不会出现问题。

npm install hexo-asset-image –save
npm install https://github.com/CodeFalling/hexo-asset-image –save

此时在markdown文件中就可以使用相对路径显示图片。

参考https://hexo.io/zh-cn/docs/asset-folders.html ，现在官网也给出了解决方案，直接在source下创建images文件夹,就可以直接使用md引用该文件夹内的图片。
